O site do Projeto Irerê é:

  • Licenciado sob uma Licença Creative Commons Atribuição 3.0 Não Adaptada.
  • Escrito em HTML5 validado por W3C.
  • Compatível com os critérios de acessibilidade WCAG 2.0.
  • Responsivo e compatível com diferentes dispositivos.
  • Construído com Bootstrap 3.0.
  • Usa as fontes Merriweather (por Eben Sorkin) e Montserrat (por Julieta Ulanovsky).
  • Gerado com Jekyll.
  • Hospedado no GitHub por meio do GitHub Pages.
  • Baseado em um template desenvolvido por Jonathan McGlone.

Este site utiliza cookies anônimos para estatísticas de acesso (Google Analytics e GoatCounter). Nenhum dado pessoal é coletado.

As fotos dos irerês têm uso com autorização dos fótografos: Lucas Martin, João Salvador, e David Cook (na ordem do menu). Sem fins lucrativos.
